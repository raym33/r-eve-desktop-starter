# Skill Drafts

Skill Forge writes generated drafts here.

Drafts are intentionally ignored by git by default. Review generated files, tests, and permission notes before copying a draft into the real `raym33/r` skill tree or committing it intentionally.

Expected draft structure:

```text
skill-drafts/<skill-name>/
  README.md
  manifest.json
  skill.py
  tests/test_skill.py
  APPROVAL.md
```
